- This module summarises what web server do behind the scenes to show you the website you want to display

# What i learned:

- If you want to connect to a site DNS finds IP adress of this site and contacts HTTP protocol to give HTML and java so you can view the site

- Load balancers are used if site has much queries and what algorithms it uses

- some of this algorithms are Round-robin sends requests to each server one by one. Weighted checks which server is least busy and sends the request there.

- web server is a software that listens for incoming connections and then utilises the HTTP protocol to deliver web content to its clients.
