# Data Encoding - THM Room

## What is this room about
How computers read and store different characters
including letters, symbols and emojis.

## What I learned
- ASCII was first — reads English letters in binary
- Many countries couldn't read their own characters
  so Unicode was created
- UTF-8, UTF-16, UTF-32 assign different byte amounts
  to each character or emoji
- More bytes = more possible characters

## What was hard
UTF-32 byte representation for emojis — 
how a single emoji maps to a specific number.
