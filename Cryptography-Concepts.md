## THM ROOM Cryptography Concepts

# What is this room about:

- explains what is cryptography

- what is plaintext and ciphertext

- what is symmetric encryption and asymmetric encryption

# What i learned

- cryptography is a method of keeping for example messages secure so no one can see private messages you send to another person by using keys

- plaintext is something you can read for example HELLO and ciphertext is encrypted text. If we encrypt HELLO it can be something like JNSIA

- symmetric encryption is an easy way of encrypting messages using 1 key beetwen sender and the receiver. Its fast but not very secure

- assymetric encryption is a more secure way of sending messages because it uses 2 keys: public and private. To encrypt a message you use receivers public key and to decrypt this a receiver uses his private key

# Relevance in cybersecurity:

- If a message is not well encrypted for example by using only symmetric encryption an attacker can intercept the message which could be a password to bank account
